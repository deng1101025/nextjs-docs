## $forceUpdate的实现

$forceUpdate的实现简单不能在简单，就是触发一下当前组件下的Watcher实例的update方法，因为这个方法就会触发整个组件的重新渲染

```js
Vue.prototype.$forceUpdate = function() {
    var vm = this; // 此时this指向就是触发这个方法的组件实例对象
    if(vm._watcher) { // 此watcher实例是在首次渲染组件的时候new Watcher() 的构造方法中创建的
        vm.watcher.update()
    }
}
```

完。。。
