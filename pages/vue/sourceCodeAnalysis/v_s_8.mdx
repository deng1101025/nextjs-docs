## 缓存函数

vue中工具方法 - 生成缓存函数，避免大量的创建新的函数导致性能损耗，所以直接把函数缓存到一个cached对象里面，存在内存中，省去了创建的时间，当然缺点就是消耗了内存，这是典型的牺牲空间降低时间损耗

```js
function cached(fn) {
    var cache = {} // 所有生成的函数在首次创建都会被缓存，都会存储到这个cahce对象里面
    return function(str) {
        var hit = cache[str]
        return hit || ( cache[str] = fn(str) )
    }
}
```
